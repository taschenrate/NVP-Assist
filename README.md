# NVP-assist

**NVP-assist** - компактный клиентский Fabric-мод для модераторской слежки в НВП. Он собирает важные сигналы по подозреваемому в один аккуратный HUD и помогает быстрее открыть меню взаимодействия с игроком.

## Возможности

- Автоматически определяет подозреваемого по сообщениям чата и actionbar.
- Работает только в режиме spectator, чтобы функции мода не мешали обычной игре.
- Показывает компактный HUD с AC / риском, AVG, reach, задержками ударов и динамической оценкой типа чита.
- Определяет варианты вроде `Reach`, `Trigger`, `Aim`, `KillAura / TargetSwitch` и их сочетания.
- Рисует точный hitbox подозреваемого, включая invis-игроков, если entity прогружена клиентом.
- Показывает луч взгляда от глаз подозреваемого: зеленый при наведении на hitbox игрока, красный при промахе.
- Анализирует резкие движения головы, большие развороты и подозрительную стабильность наводки.
- Отображает последние 7 задержек успешных ударов минималистичной полосой.
- Скрывает античит-голограммы рядом с подозреваемым и переносит нужные значения в HUD.
- Автоматически убирает HUD после бана, пропажи подозреваемого или выхода из spectator.
- Позволяет двигать HUD мышкой и менять масштаб, в том числе при открытом чате.
- Поддерживает ModMenu: настройки HUD, бинды, включение авто-TPO и базовые параметры отображения.
- По бинду может сделать `/tpo <ник>`, повернуться на hitbox подозреваемого и нажать ПКМ для открытия меню взаимодействия.
- Может автоматически телепортироваться к подозреваемому, если он дальше 50 блоков. По умолчанию выключено.

## Управление

Бинды настраиваются в стандартных настройках управления Minecraft или прямо через ModMenu в разделе **Бинды NVP-assist**.

Основные действия:

- редактирование HUD;
- включение/выключение HUD;
- ПКМ по подозреваемому;
- ReplayMod-клип, если интеграция включена.

## Требования

- Minecraft `1.20.1`
- Fabric Loader `0.14.21+`
- Fabric API `0.83.0+1.20.1`
- Java `17+`
- ModMenu `7.0.0+` опционально, но рекомендуется для удобной настройки
